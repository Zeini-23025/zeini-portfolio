# Portfolio de Développeur Web

Ce projet est un portfolio web personnel, moderne et responsive, créé avec HTML, CSS et JavaScript. Il a été conçu pour présenter mes compétences, mes projets et mon expérience professionnelle.

## ✨ Fonctionnalités

- **Design Moderne & Minimaliste** : Une interface sobre et professionnelle.
- **Responsive** : Entièrement adaptable aux ordinateurs, tablettes et mobiles.
- **Thème Dark/Light** : Un interrupteur pour basculer entre un mode clair et un mode sombre.
- **Animations Douces** : Des transitions et des effets au défilement pour une expérience utilisateur dynamique.
- **Structure Claire** : Sections dédiées pour l'accueil, les compétences, les projets, les stages et le contact.
